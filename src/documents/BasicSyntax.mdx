# 📝 MDX 基础语法演示

MDX 结合了 **Markdown** 的简洁性和 **React** 的强大功能。

## 🔤 文本格式

你可以使用标准的 Markdown 语法：

- **粗体文本**
- *斜体文本*
- `行内代码`
- ~~删除线~~

## 📋 列表和引用

### 无序列表
- 项目 1
- 项目 2
  - 子项目 A
  - 子项目 B

### 有序列表
1. 第一步
2. 第二步
3. 第三步

### 引用
> 💡 这是一个重要提示：MDX 让文档编写变得更加灵活和强大。

## 💻 代码块

```javascript
// JavaScript 代码示例
function greetUser(name) {
  return `你好, ${name}！欢迎使用 MDX。`;
}

const message = greetUser('开发者');
console.log(message);
```
